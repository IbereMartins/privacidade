# Política de Privacidade – Bot Educacional P.A.S.M.E.M.

## 1. Quem é o responsável por tratar os seus dados?

Este bot educacional (**P.A.S.M.E.M., integrado ao ChatGPT**) não possui servidor próprio, banco de dados, nem qualquer sistema independente de armazenamento de informações de usuários.
Todas as interações realizadas com o bot acontecem dentro da plataforma da **OpenAI**, que é a responsável por tratar os dados pessoais inseridos nas conversas, nos termos da sua própria Política de Privacidade.

## 2. Quais dados são tratados?

Quando você utiliza este bot, o conteúdo da sua conversa (perguntas, textos, comandos, etc.) é enviado aos serviços da **OpenAI** para que o modelo de IA gere as respostas.
Eu, como desenvolvedor do bot, **não coleto, não armazeno e não crio qualquer base de dados própria** com essas informações.

Os dados podem ser registrados e processados pela OpenAI para fins como prestação do serviço, segurança, monitoramento de abusos e melhoria dos modelos de IA, de acordo com a política da própria empresa.

## 3. Uso da API ENEM.DEV

A integração com a API pública **ENEM.DEV** é utilizada apenas para consultar provas e questões do ENEM.
Nenhum dado pessoal seu é enviado para a API ENEM.DEV: os pedidos feitos a essa API incluem apenas parâmetros técnicos (como ano da prova, disciplina, número da questão etc.), sem informações de identificação do usuário.

## 4. Política de Privacidade da OpenAI

O tratamento dos seus dados (incluindo eventuais registros técnicos e conteúdo das interações) é realizado pela **OpenAI**.
Para saber exatamente quais dados são coletados, para quais finalidades, por quanto tempo e quais são os seus direitos, consulte diretamente a Política de Privacidade da OpenAI aplicável à sua região ("Rest of World").

Você pode acessar essas informações na Política de Privacidade da OpenAI, disponível em:

[https://openai.com/pt-BR/policies/row-privacy-policy/](https://openai.com/pt-BR/policies/row-privacy-policy/)

Em caso de dúvidas sobre o uso dos seus dados pela OpenAI, recomendamos consultar a política acima e utilizar os canais de contato indicados pela própria OpenAI.
